    Fortschritt sichern

    So funktionieren Export und Import

    Deine Eingaben bleiben normalerweise nur in diesem Browser auf diesem Gerät gespeichert. Mit einer Sicherungsdatei kannst du deinen Lernstand aufbewahren oder in diesem Lernraum wiederherstellen.

    1

    Export = Lernstand sichern

    Wähle oben Export. Dein Browser lädt eine kleine JSON-Datei mit dem Namen weggesperrt-fortschritt-….json herunter. Auf dem iPad findest du sie meist unter Downloads in der App „Dateien“.

    2

    Import = Lernstand wiederherstellen

    Wähle oben Import und öffne deine zuvor exportierte JSON-Datei. Nach erfolgreicher Prüfung wird die Seite neu geladen und der gesicherte Lernstand erscheint.

    Wichtig: Eine Sicherungsdatei enthält den Lernstand aller drei Niveaustufen dieses Lernraums. Bewahre sie an einem Ort auf, den du später wiederfindest. Die Datei wird nicht an einen Server übertragen.
